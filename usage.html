Usage 

How to use MediaManager
After downloading the source code for MediaManager, you can cd into the MediaManager directory and do
$ bin/mmanager -h
for a list of available commands and a brief description about what each command does.

$ bin/mmanager -h
Usage: mmanager COMMAND [FLAGS] [ARGS]
 
Options for -h command:
 
The developers have graced you with these available commands:
    config                           Allows configuring via Irb, saving to the config file
    analyze                          Tries to guess the imported media's metadata.
    remap                            Creates a Library directory from your media files
    import                           Imports a file or directory into the library
    hasher                           Reads the paths, and generates hashes for the files
 
Common options:
    -V, --version                    Show the version number
    -h, --help                       Display ze help
